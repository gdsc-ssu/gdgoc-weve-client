## Weve client

### Commit Rule

| **태그**     | **설명**                                                 |
| ------------ | -------------------------------------------------------- |
| **Init**     | 프로젝트 생성                                            |
| **Design**   | CSS 작업                                                 |
| **Feat**     | 기능 추가                                                |
| **Fix**      | 버그 수정                                                |
| **Docs**     | 문서 수정                                                |
| **Style**    | 코드 formatting, 세미콜론(;) 누락, 코드 변경이 없는 경우 |
| **Refactor** | 코드 리팩토링                                            |
| **Test**     | 테스트 코드, 리팩토링 테스트 코드 추가                   |
| **Minor**    | 사소한 변화                                              |
